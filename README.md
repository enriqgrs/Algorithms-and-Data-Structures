# Algorithms-and-Data-Structures

Repositorio dedicado a la implementación y análisis de estructuras de datos fundamentales y paradigmas algorítmicos. El proyecto se enfoca en la eficiencia computacional (Complejidad Temporal y Espacial) y la resolución óptima de problemas lógicos.

### Contribuciones y Contenido:

El repositorio se organiza según la complejidad y el propósito de los componentes desarrollados:

- **Estructuras de Datos Lineales y No Lineales**: Implementación de estructuras fundamentales como Listas Enlazadas, Árboles Binarios de Búsqueda (ABB) y Tablas Hash, optimizando las operaciones de inserción, borrado y búsqueda.
- **Algoritmos Ávidos (Greedy)**: Desarrollo de soluciones para problemas de optimización local donde la elección óptima inmediata conduce a una solución global eficiente.
- **Programación Dinámica**: Resolución de problemas complejos mediante la descomposición en subproblemas y el uso de técnicas de memorización para reducir la carga computacional.
- **Algoritmos de Búsqueda y Ordenación**: Implementación y comparativa de eficiencia en diversos escenarios de carga de datos.

### Donde se analiza:

- **Análisis de Complejidad (O(n))**: Evaluación del rendimiento teórico y práctico de cada implementación para garantizar la escalabilidad de las soluciones.
- **Gestión Eficiente de Memoria**: Uso de punteros y estructuras dinámicas para minimizar el *overhead* en sistemas con recursos limitados.
- **Paradigmas de Resolución**: Estudio sobre cuándo aplicar Programación Dinámica frente a aproximaciones Greedy basados en la estructura del problema.
